在无参数调用函数的地方要特别小心，可别忘了加上括号。

你还可以使用参数槽（splats）接收多个参数，使用...表示: 

{% highlight javascript %}
sum = (nums...) ->
  result = 0
  nums.forEach (n) -> result += n
  result 
{% endhighlight %}
